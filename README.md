## 📦 Featured Repositories

### 🔴 Core Projects

#### **[Local-Model](https://github.com/XenoCoreGiger31/Local-Model)** — Halo v0.1.0

**Autonomous Penetration Testing Agent**

An intelligent, autonomous agent for coordinated penetration testing. Runs **Gemma 4 12B Abliterated** locally via LM Studio, orchestrates **22 integrated security tools** via MCP server, maintains persistent memory, learns from negative experiences, and generates comprehensive pentest reports.

**Key Features:**

- ✅ **Autonomous tool orchestration** — intelligent recon-to-exploit workflows (masscan → nmap → exploit chains)
- ✅ **Persistent negative experience cache** — prevents redundant attempts, optimizes testing
- ✅ **JSONL-backed short-term memory** — decision tracking, learning-based strategy refinement
- ✅ **HTML pentest report generation** — professional, actionable findings
- ✅ **22 integrated security tools** — comprehensive attack surface coverage
- ✅ **Tested against Metasploitable** — 14/24 ports successfully breached

**Tech Stack:** Python, Flask, Gemma 4 12B Abliterated, LM Studio, MCP Server, Kali Linux

**Status:** v0.1.0 Released | Production-Ready | Actively Maintained
